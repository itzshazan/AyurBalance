# 🌿 AyurBalance

AyurBalance is an AI-powered health and nutrition application designed to help users maintain a balanced lifestyle using Ayurvedic principles. The app analyzes food habits and provides personalized insights for better health.

---

## 🚀 Features

* 🥗 Smart Food Analysis
* 📷 Camera-based Food Recognition (planned)
* 📊 Health Insights Dashboard
* 🤖 AI-based Recommendations
* 📱 Clean and User-Friendly UI

---

## 🛠️ Tech Stack

* **Frontend:** Android (Kotlin / XML)
* **Backend:** (Planned / Supabase integration)
* **Database:** Supabase / Firebase (optional)
* **AI Integration:** (Future scope)

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/itzshazan/AyurBalance.git
```

2. Open in Android Studio

3. Sync Gradle and run the app
